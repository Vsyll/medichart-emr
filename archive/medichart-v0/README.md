# Arsip MediChart v0

> **Folder ini adalah arsip dari versi lama (v0) aplikasi MediChart EMR.**

## Tentang Arsip Ini

Folder `archive/medichart-v0/` menyimpan snapshot versi awal MediChart — Sistem Catatan Pemeriksaan Dokter — sebelum pengembangan ke versi 2 (v2) dimulai.

Konten arsip ini **tidak lagi diperbarui** dan hanya disimpan sebagai referensi historis.

## Catatan

- Versi ini merupakan prototipe berbasis satu file HTML dengan data in-memory (tanpa backend/database nyata).
- Untuk versi terbaru, lihat direktori utama proyek.
- Arsip ini dibuat sebagai bagian dari proses migrasi ke **MediChart EMR v2**.
